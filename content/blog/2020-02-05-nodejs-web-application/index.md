Nodejs 라이브러리를 활용
DB는 MYSQL 활용
Nodejs 중에서 express를 활용하여 서버제작
ejs 엔진을 활용

Node.js에서 es6 문법을 적용하여 모듈을 import 하기 위해서는 babel이 필요하다

npm 은 WARN도 자주 뜨고 불안정함 좀더 정확히 깔끔하게 빠르게 설치하면 좋다.
그러기위해서 yarn을 사용한다.
npm install --global yarn

yarn은 파워쉘로 사용이 안되더라
